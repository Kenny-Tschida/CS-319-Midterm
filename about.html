    This project was developed for the computer science class COM S 319: Construction of User Interfaces at Iowa State University as a midterm Project. The developers behind this project are Kenneth Tschida and Jacob Lehrman. Both put many hours and effort into this project learning how to work between HTML, CSS, and JavaScript. Kenneth Tschida specialized in creating the unifrom CSS style, home page, and JSON descriptions and images for each hike, alongside learning how to use bootstrap in unison with JavaScript. Similarily, Jacob Lehrman also worked with using bootstrap in unison with JavaScript, as well as establishing the CSS structure of the individual hike pages, alongside setting up and utilizing the google maps API.
